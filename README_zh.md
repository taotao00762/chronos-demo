Chronos - 以生活事件为驱动的学习伴侣
Chronos 是一款人工智能驱动的学习系统，其核心功能是“生活事件干扰”。它鼓励用户在学习过程中实时表达自身状态（例如疲劳、忙碌或情绪波动）。系统会将这些“抱怨”解读为生活事件信号，从而使“指导决策系统”能够实时调整学习计划，并在启动时提供一份清晰易懂的每日简报。

精彩片段
生活事件驱动：将“现实生活中的干扰”转化为可计算的学习信号。
校长决策系统：整合学习记录、每周计划和生活事件，动态调整每日日程安排。
可解释的调整：每日弹出窗口解释每次调整背后的原因，允许用户接受或修改。
特征
每日简报弹窗：显示昨日概况和今日建议。
计划模块：根据压力水平和目标生成每日任务。
AI导师：支持多会话聊天，提供实时回复和基于图像的教学。
记忆系统：存储偏好、事件和技能记忆，并支持搜索和更新。
知识图谱：根据学习记录生成概念网络。
设置中心：语言、学习偏好、模型配置等。

入门
安装依赖项
pip install flet google-genai aiosqlite numpy pydantic pyvis pytest
配置 API 密钥（选择一个）

环境变量：GEMINI_API_KEY=your_key
或者在“设置”页面中输入您的 Gemini API 密钥。
运行应用程序

python main.py
重要提示
该项目会在data/本地生成数据库和服务器文件。
不要将data/*.db、data/images/或提交data/settings.json到公共存储库。
该程序.gitignore已预先配置为过滤掉这些本地敏感文件。
ReMe 相关功能是可选依赖项——即使没有它们，核心工作流程也能运行。
